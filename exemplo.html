📋 Como Usar

Para usar o player em seu site:

  1. Coloque os arquivos de áudio na pasta /audio/nome_da_musica/
  2. Use o iframe com os atributos data-musica, data-width e data-height
  3. Ajuste as dimensões width e height do iframe conforme necessário

🎮 Controles Principais

Play: Inicia todas as faixas
Pause: Pausa reprodução
Stop: Para e reinicia

🎛️ Controles por Faixa

Mute: Silencia faixa específica
Solo: Isola uma faixa
Volume: Controle individual

⏱️ Navegação

Barra de Progresso: Clique para pular
Tempo: Exibe atual/total
Sincronia: Mantém todas as faixas alinhadas

🔧 Configuração

data-musica: Nome da pasta
data-width: Largura do player
data-height: Altura do player

🎵 Formatos Suportados

O player suporta os seguintes formatos de áudio:

  • MP3 - Mais compatível
  • WAV - Qualidade superior
  • OGG - Código aberto
  • M4A - Apple/iTunes
